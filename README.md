## Arab Collegiate

Regional ICPC-style contests from Arab countries.

- ACPC 2018 (Arab Collegiate Programming Contest)
- MCPC 2025 (Moroccan)
- LCPC 2025 (Lebanese)
- PCPC 2025 (Palestinian)
- TCPC 2026 (Tunisian)
- MNPC 2023 (Moroccan)
